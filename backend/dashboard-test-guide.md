# 在线考试系统 - 仪表板测试指南

## 概述
本指南用于测试新增的用户仪表板系统，包括教师仪表板和学生仪表板的功能测试。

## 测试前准备

### 1. 启动后端服务
确保Spring Boot后端服务已启动：
```bash
cd backend
mvn spring-boot:run
```

### 2. 初始化测试数据
执行以下SQL脚本初始化测试数据：
```bash
mysql -u root -p online_exam < test-data.sql
```

### 3. 打开测试页面
在浏览器中打开测试页面：
```
http://localhost:8080/test-dashboard.html
```

## 测试步骤

### 1. 登录功能测试
1. 在测试页面中，使用教师账号进行登录测试
2. 使用学生账号进行登录测试
3. 测试错误密码登录
4. 测试不存在的用户名登录
5. 测试角色选择错误的情况

### 2. 仪表板页面测试
1. 以教师身份登录后，打开教师仪表板
2. 以学生身份登录后，打开学生仪表板
3. 检查仪表板页面是否正常加载
4. 检查页面上的模拟数据是否正确显示

### 3. 权限控制测试
1. 测试未登录状态下访问仪表板
2. 测试教师身份访问学生仪表板
3. 测试学生身份访问教师仪表板
4. 测试登录状态检查功能

### 4. 退出登录测试
1. 测试退出登录功能
2. 退出后尝试访问需要权限的页面
3. 检查本地存储是否已清除

### 5. 模拟数据测试
1. 检查教师仪表板上的模拟数据（创建的考试数量、学生总数等）
2. 检查学生仪表板上的模拟数据（已参加考试、平均分数等）

## 常见问题

### 1. 登录失败
- 检查后端服务是否启动
- 检查数据库连接是否正常
- 确认用户名和密码是否正确
- 确认角色选择是否正确

### 2. 仪表板页面无法加载
- 检查前端文件路径是否正确
- 检查浏览器控制台是否有错误信息
- 确认是否已正确登录

### 3. 权限控制失效
- 检查Session管理是否正常
- 确认前端权限检查逻辑是否正确
- 检查后端权限控制接口是否正常
